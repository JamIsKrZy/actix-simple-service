Create User (Employee/Admin Previlage)

post
https://127.0.0.1:8080/admin/user/new

🔸 Request body (Json):

{
    "email": String,
    "password": String,
    "username": String,
    "first_name": String,
    "last_name": String,
    "phone_no": String,
    "location"?: String,
    "role:" "admin" | "employee" |"regular"
}

🔹 Response body (Json)

{

    // successful service
    "success"?: {
        "query": 
    },
    
    // fail to comply service
    "fail"?: {
        "message": String
    },

    // Any internal service problem
    "error"?: {
        "message": String
    }

}

Create Bundle

post
https://127.0.0.1:8080/admin/bundle/new

🔸 Request body (Json):

{
    "name": String,
    "price": number,
    "status"?: "unavailable" | "available",
    "products": [
        {
            "product_id": numbers,
            "quantity": number
        }
    ]
}

🔹 Response body (Json)

{
    "status": "success" | "fail" | "error",
    "message": String
}

Edit Bundle

patch
https://127.0.0.1:8080/admin/bundle/edit/{bundle_id}

🔸 Path:

NameType
bundle_idstring

🔸 Request body (Json):

{
    "name"?: String,
    "description"?: String,
    "price": number,
    "stocks": BigInt,
    "remove_product"?: [ 
        {
            "product_id": number
        }
    ],
    "add_product"?: [
        {
            "product_id": numbers,
            "quantity": number
        }
    ],
    "edit_product"?: [
        {
            "product_id": numbers,
            "quantity": number
        }
    ]
}

Info: "edit_product" does only change the quantity number, not the product itself

🔹 Response body (Json)

{
    "status": "success" | "fail" | "error",
    "message": String
}
